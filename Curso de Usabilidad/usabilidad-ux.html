Proyecto del curso
Prueba de usabillidad documentada

Aprende a usar la documentación necesaria para llevar a cabo una prueba de usabilidad presencial o remota de un producto. Entenderás cuáles son las mejores prácticas para implementar, aprenderás a usar el Kit de Usabilidad que incluye toda la documentación necesaria para registrar los hallazgos que tus usuarios te comparten durante la prueba.
